### E4-3.py
关于numpy数组切片赋值操作，复制的是指针，那应该只是影响上一次操作，如果切片赋值2次，原本的数组应该不会变化。具体测试在代码中有。  
两次slice操作之后赋值不会影响到原数数组，其实是是只影响最近的一次切片操作。
